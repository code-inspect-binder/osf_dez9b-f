The files here available contain indications for implementing Bayesian informative hypotheses testing with ANOVA ("BaIn_ANOVAinJASP.jasp" and "BaIn_ANOVAinR.R") or regression models ("BaIn_REGRESSIONinJASP.jasp" and "BaIn_REGRESSIONinR.R") following 5 simple steps, in JASP and R software, respectively.

The file “dataset_anova.txt” contains the dataset used for ANOVA examples.

The file “dataset_regression.txt” contains the dataset used for REGRESSION examples.

The file “creating the datasets.R” contains the R script used to create the two datasets.
